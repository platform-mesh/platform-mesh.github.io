# Kubernetes GraphQL gateway

## Purpose

The Kubernetes GraphQL gateway exposes Kubernetes Resource Model APIs through GraphQL for the portal and other clients.

## Runtime role

It gives UI clients a typed query layer over kcp and Platform Mesh resources while preserving the underlying KRM model.
