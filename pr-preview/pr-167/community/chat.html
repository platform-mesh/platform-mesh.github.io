Chat ​

Platform Mesh chat lives on the Linux Foundation Zulip server. Two channels cover most conversations.

Channels ​

ChannelUse it for
neonephos-platform-mesh-discussionDesign discussions, RFCs, and general questions about Platform Mesh.
neonephos-platform-mesh-supportHelp with installing, running, or operating Platform Mesh.

Don't report security issues in chat ​

Posting a vulnerability in public chat exposes it to potential attackers before a fix exists. Follow the security policy to report it through the project's private intake.

Code of conduct ​

The Code of Conduct applies in all channels.
