IAM UI ​

Purpose ​

The IAM UI is the Platform Mesh user interface for managing identities, roles, and authorization relationships. It surfaces inside the Platform Mesh Portal as the primary IAM administration experience.

Runtime role ​

It calls the IAM service to read and write authorization state, presents accounts and their members, and exposes the operations needed to grant or revoke access in a Platform Mesh deployment.
